## ✨ Новые возможности

### 🖼️ Улучшенная галерея миниатюр (Desktop)
- Миниатюры размещены прямо под основным изображением
- Компактный размер (40x40px вместо 80x80px)
- Живое превью с применёнными рецептами
- Центрирование галереи как у панели инструментов
- Автоматический скролл к активной миниатюре

### 📱 Улучшенный мобильный интерфейс
- Кнопки действий ("Apply to all" и "Export") перенесены вниз экрана
- Кнопки расположены рядом с равной шириной
- Более удобный доступ при работе большим пальцем
- Чистый список пресетов без лишних элементов

### ⚡ Индикатор загрузки
- Показывается при применении пресета ко всем изображениям
- Информативное сообщение с количеством обрабатываемых фото
- Предотвращает множественные клики во время обработки

### 🎯 Кнопка "Apply to all" на панели инструментов (Desktop)
- Перенесена из боковой панели на главную панель инструментов
- Расположена после кнопки с выбранным рецептом
- Показывается только при загрузке 2+ изображений

### 🔄 Исправлена синхронизация
- Переключение между изображениями обновляет и превью, и панель рецептов
- Миниатюры показывают актуальное состояние с применёнными эффектами
- Плавная работа при быстром переключении

## 🚀 Инструкция по деплою

### Быстрый старт (5 минут)

1. **Скачайте файлы**
   ```bash
   # Вариант 1: Клонируйте репозиторий
   git clone <repo-url>
   cd Photochrome

   # Вариант 2: Используйте готовый архив
   # Файл: photochrome-v1.2.0.tar.gz
   ```

2. **Загрузите на сервер**
   ```bash
   # Через FTP/SFTP
   # Загрузите всё содержимое папки out/

   # Или через SSH
   scp -r out/* user@server:/path/to/web/root/

   # Или распакуйте архив на сервере
   tar -xzf photochrome-v1.2.0.tar.gz -C /path/to/web/root/
   ```

3. **Проверьте работу**
   - Откройте https://photochrome.netdesigner.ru
   - Загрузите тестовые изображения
   - Проверьте применение пресетов
   - Проверьте экспорт

### Подробная инструкция
См. файл `out/DEPLOY.md` для детальной информации.

## 📞 Поддержка

### При возникновении проблем:
1. Проверьте Browser Console на ошибки
2. Убедитесь, что все файлы загружены
3. Проверьте права доступа к файлам
4. Убедитесь, что mod_rewrite включен (Apache)
5. Проверьте, что CORS настроен правильно

### Известные особенности:
- Обработка больших изображений может занимать время
- Violations в консоли - это нормально (см. документацию)
- Яндекс.Метрика блокируется AdBlock'ами
