# Roadmap

## Реализовано

### Движок обработки
- **10 плёночных симуляций**: Provia, Velvia, Astia, Pro 400H, Superia, Acros, Neopan, Eterna, Classic Chrome, Classic Neg
- **Dynamic Range**: DR100 / DR200 / DR400 (подъём теней, компрессия светов)
- **White Balance**: 6 пресетов — Auto, Daylight, Shade, Cloudy, Tungsten, Fluorescent
- **Тонкая настройка**: Highlight/Shadow, Color, Sharpness, Clarity, WB Shift, Grain, Color Chrome, Color Chrome FX Blue
- **WebGL-ускорение**: GPU-шейдер для фото (отдельный от видео пайплайн)
- **Web Worker**: асинхронный экспорт без блокировки UI, zero-copy передача через ArrayBuffer

### Коллекция рецептов
- **51 рецепт**, сгруппированных по симуляции (10 групп) и сценарию использования (Portrait, Landscape, Street, Cinema, B&W, Everyday)
- Live preview рецептов на загруженном фото

### Редактор
- Поворот (90° CW / CCW)
- Crop с 5 соотношениями сторон (1:1, 4:3, 3:4, 16:9, 9:16) и перетаскиванием рамки
- Рандомизация настроек

### Экспорт
- JPEG с качеством 0.95
- EXIF метаданные — название и параметры рецепта сохраняются в файле
- Водяной знак

### Видео
- Применение симуляций к видео до 30 секунд
- WebGL для GPU-рендеринга кадров, mp4-muxer для кодирования
- Safari fallback через MediaRecorder

### Платформа
- PWA manifest с поддержкой share_target (принимает JPEG/PNG/WebP)
- Клиентская обработка — фото не покидает устройство

## Ближайшее

### 1. Новые симуляции Fujifilm

Камеры последних поколений (X-Pro3, X-T5, X100VI) получили симуляции, которых нет в текущей базе:

| Симуляция | Камера | Характер |
|---|---|---|
| **Nostalgic Neg** | X-T5 / GFX | Тёплые, выгоревшие тона, слабый контраст |
| **Reala Ace** | X100VI / X-T50 | Нейтральная, точная цветопередача кожи |
| **Eterna Bleach Bypass** | X-Pro3 | Высококонтрастный, десатурированный |
| **Pro Neg. Hi** | X-Pro2 и новее | Насыщенный портретный, чистые цвета |
| **Pro Neg. Std** | X-Pro2 и новее | Мягкий портретный, пастельный тон |

Каждая симуляция — JSON в `src/presets/simulations/` с тональной кривой, цветовым балансом и базовой насыщенностью.

### 2. Расширение коллекции рецептов

Пополнить базу рецептами из открытых источников ([Fujix Weekly](https://fujixweekly.com), Film Fridays, сообщество Fujifilm X):

- Минимум 2–3 рецепта под каждую новую симуляцию
- Закрыть пустые сценарии у существующих симуляций (например, Everyday у Velvia)

### 3. PWA офлайн-режим

Manifest и метатеги уже настроены, но service worker отсутствует — приложение не работает без интернета. Добавить через `vite-plugin-pwa` (Workbox):

- Кэширование app shell, JS/CSS бандлов и JSON рецептов при первом запуске
- Полноценная работа в режиме самолёта после установки
- Стратегия обновления: уведомление пользователя при выходе новой версии
